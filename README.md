## 🌟 เกี่ยวกับโปรเจกต์ (About)
**CarCare Pro** คือเว็บแอปพลิเคชัน (PWA) ที่ออกแบบมาเพื่อช่วยคนรักรถในการจัดการค่าใช้จ่าย บำรุงรักษา และเก็บสถิติรถยนต์ทุกคันอย่างเป็นระบบ รองรับรถยนต์ทั่วไป, กระบะ, มอเตอร์ไซค์, และ **ยานยนต์ไฟฟ้า (EV)** ข้อมูลทั้งหมดถูกเชื่อมต่อผ่าน Cloud (Firebase) ทำให้ปลอดภัยและดูผ่านอุปกรณ์ใดก็ได้ตลอดเวลา

🌟 **ในอัปเดต 1.0.1 นี้** เราได้เพิ่มสเน่ห์ความสนุกด้วยฟีเจอร์ใหม่ **"🎮 Pixel Quest"** เปลี่ยนแดชบอร์ดธรรมดาให้กลายเป็นหน้าจอเกม RPG แบบ retro pixel สุดน่ารัก!

---

## ✨ ฟีเจอร์เด่น (Key Features)

### 🎮 Pixel Quest Dashboard *[ใหม่ใน v1.0.1!]*
- เฝ้าดูการเติบโตของ "อาณาจักรรถยนต์" ของคุณบนหน้าจอแดชบอร์ดสไตล์เกม Retro RPG ระบบ 8-bit
- อาคารในเกม (Garage, Gas, Charge) สามารถกดเพื่อข้ามไปบันทึกข้อมูลรถได้ทันที
- คำนวณค่า **LV**, **HP bar**, และ **XP bar** อิงจากการแวะเข้าใช้บริการของคุณ
- โชว์ **Quest Log** (ประวัติล่าสุด) และเตือน **Upcoming Quests** (นัดหมายที่ใกล้ถึง) 

### 📊 แดชบอร์ดสรุปผลอัจฉริยะ (Dashboard)
- หน้าจอสรุปค่าใช้จ่ายแบบจริงจังสำหรับวิเคราะห์ แยกเป็นหมวดหมู่ (ซ่อมบำรุง, น้ำมัน, ชาร์จ EV)
- ดีไซน์สวยงามสบายตา ออกแบบมารองรับ **Premium Dark Mode** และ **Light Mode**

### 🚘 การจัดการยานพาหนะ (Vehicle Management)
- เพิ่ม/ลบ รายการรถยนต์ได้ไม่จำกัดจำนวน
- อัปโหลดรูปภาพประจำรถได้

### 🛠️ บันทึกประวัติการซ่อมบำรุง (Maintenance Records)
- บันทึกงานซ่อม เปลี่ยนอะไหล่ เปลี่ยนของเหลว ถ่ายน้ำมันเครื่อง แบตเตอรี่ ยาง ฯลฯ
- ตั้งนัดหมายการเข้าซ่อม/เช็คระยะรอบถัดไป พร้อมระบบแจ้งเตือนจำนวนวันที่เหลือ
- ส่งออกรายงานเป็นไฟล์ PDF และ CSV 

### ⛽ บันทึกการเติมน้ำมัน (Fuel Log) 
- บันทึกการเติมน้ำมันและแสดงกราฟวิเคราะห์ 
- ระบบจะคำนวณอัตราสิ้นเปลืองน้ำมัน (กิโลเมตร/ลิตร) ทันที 

### ⚡ แพลตฟอร์มสำหรับรถ EV (EV Charging Log)
- บันทึกประวัติการชาร์จแบตเตอรี่ (เลือกระหว่างชาร์จ AC หรือ DC และระบุตู้ชาร์จได้)
- ระบบจะคำนวณประสิทธิภาพการวิ่งของรถไฟฟ้า (กิโลเมตร/kWh) อย่างแม่นยำ

### 🔒 ความปลอดภัย (Security)
- สมัครและล็อคอินเข้าใช้งานง่ายๆ ด้วยบัญชี Google Authentication
- มีระบบปกป้องข้อมูลหน้าเครื่องด้วยรหัส PIN (เปิดแอปต้องใส่ PIN 6 หลัก)

## 📱 วิธีการเริ่มต้นใช้งาน (How to Use)
1. กดที่ [ลิงก์ใช้งานจริง](#-เข้าใช้งานจริงได้ที่) และล็อคอินด้วยบัญชี Google
2. (ทางเลือก) ตั้งรหัส PIN 6 หลักในการเข้าใช้งาน
3. กดเมนู **"รถของฉัน"** (ซ้ายมือ) และกดเพิ่มรถคันใหม่ก่อนเลย
4. เริ่มสร้างประวัติให้รถของคุณ โดยไปที่ **"บันทึกการซ่อม"**, **"บันทึกการเติมน้ำมัน"**, หรือ **"บันทึกการชาร์จ EV"**
5. ตรวจสอบสถานะรถสนุกๆ แบบเกมที่แท็บ **"🎮 Pixel Quest"** หรือสถิติรวมที่หน้า **"แดชบอร์ด"**

## 🛠️ โครงสร้างโปรเจกต์ (Project Structure)
โปรเจกต์คลาสสิก ปรับแก้ง่าย สไตล์ Vanilla Web Stack
- **index.html** - โครงสร้างแอปพลิเคชัน
- **styles.css** - จัดการสไตล์ (Dark/Light Mode) รวมถึง Pixel Art Animations ทั้งหลาย
- **app.js** - โลจิคทั้งหมดดึงผ่าน Firebase และประมวลผลบน Client แบบ SPA
- Firebase config & rules - `firestore.rules`, `firebase.json`
