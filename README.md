## 🌟 เกี่ยวกับโปรเจกต์ (About)
**CarCare Pro** คือเว็บแอปพลิเคชัน (PWA) ที่ออกแบบมาเพื่อช่วยคุณบันทึกและจัดการค่าใช้จ่ายเกี่ยวกับรถยนต์ทั้งหมดแบบครบวงจร รองรับตั้งแต่รถยนต์คันเก่ง รถกระบะ มอเตอร์ไซค์ ไปจนถึง **ยานยนต์ไฟฟ้า (EV)** ด้วยระบบ Cloud (Firebase) ข้อมูลของคุณจะปลอดภัยและสามารถเข้าถึงได้ทุกที่ทุกเวลา

## ✨ ฟีเจอร์เด่น (Key Features)

### 📊 แดชบอร์ดสรุปผลอัจฉริยะแบบ Real-Time (Dashboard)
- หน้าจอสรุปค่าใช้จ่ายแยกเป็นหมวดหมู่ (ซ่อมบำรุง, เติมน้ำมัน, ชาร์จ EV) เห็นภาพรวมแบบรวดเร็ว
- แสดงรายการกิจกรรมล่าสุด และการนัดหมายที่กำลังจะมาถึงเพื่อเตือนความจำ
- การ์ดรายงานที่ออกแบบมาในสไตล์ Premium Dark Theme อ่านง่าย สบายตา 

### 🚘 การจัดการยานพาหนะ (Vehicle Management)
- เพิ่ม/แก้ไขรายชื่อรถยนต์ที่คุณเป็นเจ้าของได้ไม่จำกัดจำนวน
- รองรับประเภทรถหลากหลาย (เก๋ง, กระบะ, มอเตอร์ไซค์, รถEV, และอื่นๆ)
- อัปโหลดรูปรถยนต์ส่วนตัวได้ เพื่อความสวยงาม

### 🛠️ ระบบบันทึกประวัติการซ่อมบำรุง (Maintenance Records)
- บันทึกรายการซ่อม, เปลี่ยนถ่ายน้ำมันเครื่อง, สลับยาง และประวัติเข้ารับบริการทั้งหมด
- บันทึกชื่ออู่/ศูนย์บริการ, เลขไมล์ ณ วันที่ซ่อม, และค่าใช้จ่าย
- ตั้งการนัดหมายในอนาคต เช่น นัดเช็คระยะล่วงหน้า
- Export รายงานค่าซ่อมบำรุงในรูปแบบ PDF และ CSV ได้อย่างสวยงาม

### ⛽ ระบบบันทึกการเติมน้ำมัน (Fuel Log) 
- บันทึกข้อมูลการเติมน้ำมัน: ราคาน้ำมัน, ลิตรที่เติม, เลขไมล์ปัจจุบัน
- **คำนวนอัตโนมัติ:** ค่าน้ำมันเฉลี่ย (บาท/ลิตร) และอัตราสิ้นเปลือง (กิโลเมตร/ลิตร) ทันที
- ดูกราฟวิเคราะห์ประสิทธิภาพน้ำมัน และกราฟเปรียบเทียบค่าใช้จ่ายรายเดือน/รายคัน

### ⚡ แพลตฟอร์มสำหรับรถ EV (EV Charging Log) *[ฟีเจอร์ใหม่!]*
- บันทึกประวัติการชาร์จแบตเตอรี่รถไฟฟ้าอย่างละเอียด (รหัสสถานีชาร์จ PTT, EA, PEA, ฯลฯ)
- เลือกระหว่างชาร์จแบบ AC (กระแสสลับ) และ DC (กระแสตรง)
- **ประมวลผลทันที:** คำนวณค่าไฟเฉลี่ยหน่วยละกี่บาท, ประสิทธิภาพการวิ่ง (กิโลเมตร/kWh)
- กราฟเฉพาะสำหรับยานยนต์ไฟฟ้า เช่น อัตราส่วนหัวชาร์จ หรือกราฟประสิทธิภาพแบตเตอรี่เทียบความประหยัด

### 🔒 ความปลอดภัย และสิทธิส่วนบุคคล (Security)
- เข้าสู่ระบบง่ายนิดเดียวผ่านบัญชี Google Authentication
- มีระบบเซฟความปลอดภัย 2 ชั้นด้วยตั้งค่า PIN Mode ส่วนตัว ป้องกันคนแอบดูข้อมูล
- ใช้งานระบบ Cloud Firestore ทำงานได้ลื่นไหลซิงค์ข้อมูลตรงทุกหน้าจอ

## 🛠️ วิธีนำโปรเจกต์ไปใช้งาน (How to Clone & Run)

หากคุณต้องการนำซอร์สโค้ดนี้ไปพัฒนาต่อหรือรันบนเครื่องคอมพิวเตอร์ของคุณเอง สามารถทำได้ตามขั้นตอนดังนี้:

### 1. การโคลนโปรเจกต์ (Clone Source Code)
เปิด Terminal หรือ Command Prompt แล้วใช้คำสั่ง:
```bash
git clone https://github.com/warchag/car-repair-tracker.git
cd car-repair-tracker
```

### 2. การเปิดใช้งาน (Running Locally)
เนื่องจากโปรเจกต์นี้ถูกพัฒนาด้วย HTML, CSS และ Vanilla JavaScript คุณสามารถรันได้ง่ายๆ ดังนี้:
- **วิธีที่ 1 (ง่ายที่สุด):** ดับเบิลคลิกไฟล์ `index.html` เพื่อเปิดผ่าน Browser ได้ทันที
- **วิธีที่ 2 (แนะนำ):** ใช้ Extension **"Live Server"** ใน VS Code หรือรันผ่าน Local Server อื่นๆ (เช่น `python -m http.server`) เพื่อให้ฟีเจอร์บางอย่างทำงานได้อย่างสมบูรณ์

### 3. การตั้งค่า Firebase (Configuration)
โปรเจกต์นี้เชื่อมต่อกับ Firebase สำหรับระบบสมาชิกและฐานข้อมูล หากคุณต้องการใช้ฐานข้อมูลส่วนตัว:
1. สร้างโปรเจกต์ใหม่ที่ [Firebase Console](https://console.firebase.google.com/)
2. เปิดใช้งาน **Authentication (Google)** และ **Firestore Database**
3. คัดลอก `firebaseConfig` จากหน้า Settings มาใส่ในไฟล์ [app.js](file:///Users/worracag/Desktop/Work/car-repair-tracker/app.js) (บรรทัดที่ 8)

## 🔒 ความปลอดภัยและสิทธิส่วนบุคคล (Security & Privacy)

**คำถามที่พบบ่อย: การนำ `apiKey` ขึ้น GitHub ปลอดภัยหรือไม่?**
- **คำตอบคือ ปลอดภัยครับ:** โดยปกติแล้ว `apiKey` ของ Firebase Web SDK ไม่ใช่ความลับ (Secret) ที่ต้องซ่อน แต่เป็นเพียง ID สำหรับระบุโปรเจกต์ ซึ่งข้อมูลฝั่ง Client-side ส่วนใหญ่จะเป็นแบบนี้อยู่แล้ว
- **สิ่งที่ปกป้องข้อมูลจริงๆ คือ [Firestore Security Rules](file:///Users/worracag/Desktop/Work/car-repair-tracker/firestore.rules):** โปรเจกต์นี้ถูกตั้งค่าให้ **"เฉพาะเจ้าของข้อมูลที่ Login เท่านั้น"** ถึงจะสามารถอ่านและแก้ไขข้อมูลของตัวเองได้ (request.auth.uid == userId)
- **ข้อควรระวัง:** ไม่ควรนำไฟล์ Service Account JSON (ฝั่ง Server-side) ขึ้น GitHub โดยเด็ดขาด (แต่ในโปรเจกต์นี้ไม่ได้ใช้ครับ)
