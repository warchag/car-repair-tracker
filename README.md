# 🚗 CarCare Pro

CarCare Pro เป็นเว็บแอปพลิเคชันสำหรับจัดการและบันทึกข้อมูลเกี่ยวกับรถยนต์ของคุณ ไม่ว่าจะเป็นรถยนต์สันดาปทั่วไปหรือรถยนต์ไฟฟ้า (EV) ช่วยให้คุณติดตามประวัติการซ่อม จัดการค่าใช้จ่าย และดูสถิติการใช้งานได้อย่างลงตัวและสวยงาม

## ✨ ฟีเจอร์หลัก (Features)

*   **🚙 จัดการข้อมูลรถยนต์**: 
    *   เพิ่ม ลบ และแก้ไขข้อมูลรถยนต์ที่ครอบครองได้หลายคัน
    *   รองรับทั้งรถยนต์ระบบน้ำมัน และรถยนต์ไฟฟ้า (EV)
*   **🔧 บันทึกประวัติการซ่อมบำรุง**: 
    *   จดบันทึกรายการซ่อม อะไหล่ ค่าใช้จ่าย วันที่ และสถานที่ 
    *   รองรับการอัปโหลดและแนบรูปภาพใบเสร็จหรือรูปประกอบการซ่อม (จำกัด 4 รูปต่อรายการ)
    *   ตั้งการแจ้งเตือนการซ่อม/เช็คระยะครั้งถัดไปล่วงหน้าได้
*   **⛽ บันทึกการเติมน้ำมัน**: 
    *   บันทึกปริมาณน้ำมัน (ลิตร) ราคา บาท/ลิตร และระยะทางล่าสุด
    *   ช่วยคำนวณอัตราสิ้นเปลือง และสรุปค่าใช้จ่ายค่าน้ำมันรายเดือน
*   **⚡ บันทึกการชาร์จไฟ (EV Charging Log)**: 
    *   ระบบออกแบบเฉพาะรองรับรถ EV เพื่อบันทึกการชาร์จทั้งแบบ AC และ DC
    *   บันทึกหน่วยพลังงาน (kWh), ราคาต่อหน่วย, และระบุสถานีชาร์จ (เช่น PTT EV Station, PEA Volta, ฯลฯ)
*   **📊 แดชบอร์ดและสถิติ (Analytics)**: 
    *   หน้าสรุปผลที่ดูง่ายมาพร้อมกราฟแสดงค่าใช้จ่ายต่างๆ (Chart.js / CSS Visualization)
    *   วิเคราะห์อัตราการสิ้นเปลืองพลังงาน และนำเสนอข้อมูลในรูปแบบที่เข้าใจง่าย
    *   สามารถส่งออก (Export) รายงานสรุปค่าใช้จ่ายต่างๆ ออกมาในรูปแบบตัวเอกสาร **PDF** ได้
*   **☁️ ระบบ Cloud Sync ซิงค์ข้อมูลข้ามอุปกรณ์**: 
    *   เชื่อมต่อฐานข้อมูลกับ Firebase Firestore ทำให้ข้อมูลถูกบันทึกขึ้นคลาวด์ ไม่สูญหายเมื่อเปลี่ยนอุปกรณ์

## 🛠 เทคโนโลยีที่ใช้งาน (Technologies)

*   **Frontend**: HTML5, CSS3 (ประยุกต์ใช้ดีไซน์สไตล์ Modern UI และ Glassmorphism), Vanilla JavaScript
*   **Backend & Database**: Firebase Firestore (NoSQL Database)
*   **Hosting**: Firebase Hosting
*   **PDF Generation**: ใช้ไลบรารีภายนอกสำหรับการพิมพ์และ Export ออกเป็นเอกสาร

## 🚀 วิธีการติดตั้งและรันโปรเจกต์ (Getting Started)

1. **เริ่มรันโปรเจกต์เบื้องต้น (Local Development)**
   * Clone โปรเจกต์ลงมาที่เครื่องของคุณ
   * สามารถเปิดไฟล์ `index.html` ผ่านเว็บบราวเซอร์ได้โดยตรงเพื่อทดสอบการแสดงผล 
   * แนะนำให้ใช้งานผ่าน Extension ของ Code Editor เช่น **Live Server** ใน VS Code เพื่อจำลองเซิร์ฟเวอร์เสมือน

2. **การตั้งค่า Firebase (Database)**
   * โปรเจกต์ปัจจุบันเปิดใช้งาน Firebase เรียบร้อยแล้ว (อ้างอิง Configuration ในไฟล์ `app.js`)
   * หากต้องการเชื่อมต่อเข้ากับฐานข้อมูลส่วนตัวของคุณเอง ให้ปรับปรุงตัวแปร `firebaseConfig` ให้ถูกต้องตรงกับโปรเจกต์ใหม่ครับ

3. **การ Deploy ขึ้นใช้งานจริงด้วย Firebase Hosting**
   * ติดตั้งไลบรารี Firebase CLI: `npm install -g firebase-tools`
   * ลงชื่อเข้าใช้ด้วยคำสั่ง: `firebase login`
   * สั่ง Deploy ขึ้นระบบจริง: `firebase deploy`

## 📝 โครงสร้างของโปรเจกต์ (Project Structure)

*   `index.html`: รวมโครงสร้างหลักของหน้าเว็บ รวมถึง UI component และ Modal ต่างๆ
*   `styles.css`: ไฟล์ปรับแต่งสไตล์ UI/UX ออกแบบโทนสี การจัดวาง การทำ Responsive รองรับทั้งมือถือและคอมพิวเตอร์
*   `app.js`: บรรจุ Logic การทำงานทั้งหมด, การบันทึกข้อมูล (CRUD), การเชื่อมกับ Firebase, ฟังก์ชันการค้นหา (Filters), และการแสดงผลสถิติ
